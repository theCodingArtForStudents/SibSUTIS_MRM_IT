Detailed Description

WAbstractItemModel Базовый класс для всех моделей описывающий основные методы работы с моделью и являющийся частью механизма Model-View реализованного в WWin.

Для реализации собственной модели, Вам необходимо унаследовать этот класс и реализовать абстрактные методы. Реализуя методы в унаследованном классе не забывайте вызывать метод dataChanhed с указанием диапозона изменившихся значений.

Класс реализует систему связывания View и Model. Для этого при регистрации модели во View, View вызывает метод модели __addUpdateListened и подписывается на прослушивание сообщений об обновлении модели. Данная система позволяет уведомлять все View к которым подключена одна модель о изменении данных.
